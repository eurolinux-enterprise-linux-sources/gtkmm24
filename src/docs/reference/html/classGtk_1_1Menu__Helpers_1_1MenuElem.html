Detailed Description

Use this class and its subclasses to build menu items.

For example,

m_Menu_File.items().push_back( Gtk::Menu_Helpers::MenuElem("_New",
sigc::mem_fun(*this, &ExampleWindow::on_menu_file_new) ) );
